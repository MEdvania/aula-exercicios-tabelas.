Funções de cada tags

TABLE : cria uma tabela de uma lista de argumentos de registro
TH : define uma célula cabeçalho do grupo de células de sua tabela.
TR : é usada para definir uma linha.
TD : As letras td significam "table data," que é o conteúdo de uma célula de dados.
THEAD : inha de cabeçalho de uma tabela HTML, é utilizado para destacar os títulos das colunas de uma tabela.
TBODY : estrutura todo o conteúdo da tabela, para que assim o navegador qual exatamente é o conteúdo da tabela.
TFOOT : é utilizada para definir conteúdos de rodapé dentro de uma tabela e juntá-lo ao corpo e ao cabeçalho.
ALIGN : é inglesa, e como se pode adivinhar, significa alinhar. E sua sintaxe vai precisar de uma informação: do local que você quer linhas.
BORDER : Esta é a maneira abreviada de você declarar todos os valores para as propriedades das bordas, ou seja, pode-se definir valores para as três propriedades das bordas
CAPTION : representa o título de uma tabela.
ROWSPAN : Especifica o número de linhas a serem ocupados por uma célula. Utilizado para expandir uma célula verticalmente, também assume o valor por defeito de 1 em todas as células.
COLSPAN : Há tabelas que uma linha (geralmente o cabeçalho e o rodapé) ocupam o espaço de todas as colunas.
WIDTH : determina a largura da área de conteúdo de um elemento. A área de conteúdo fica dentro do preenchimento, da borda, e da margem de um elemento.
